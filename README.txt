## Execution: Compile code for ATMega8 chip in Atmel Studio, and program the chip.
Pins to be connected as below

  PB1 : PWM1 Redlight count
  PB2 : PWM2 Last speed
  PB3 : LED0 : Red light
  PB4 : LED1 : Yellow light
  PB5 : LED2 : Green light

  PC0 : POT1 : Period select
  PC2 : LED3 : Red light flash
  PC3 : LED4 : Configmode flash
  PC4 : speed timing pin

  PD2 : SW5 : LB1
  PD3 : SW6 : LB2
  PD4 : SW7 : LB3
  PD7 : SW0 : Config switch
